## Overview
**MultiEFM** (Multi-study Efficient Factor Model) 是一个专门用于处理多研究/多批次数据的统计分析工具包。本项目通过高效的因子模型实现多维数据的整合与特征提取。

本项目是我的学士学位论文（或研究课题）的核心代码实现，结合了 C++ (Rcpp) 加速，旨在提供稳定且高效的统计计算支持。
